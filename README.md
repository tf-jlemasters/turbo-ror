# **Turbo Message Board**
## Description 
A friends list application where you can create, update, view, and delete messages from the message board. Used to learn how to interact with turbo in Rails 7

## Installation
Fork repository, and open the project in your terminal. Install necessary gems and enter "rails s" in your console to begin the server
